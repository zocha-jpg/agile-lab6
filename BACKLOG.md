# Product Backlog

## User Stories

- **US-01** Jako uzytkownik chce dodawac zadania, aby sledzic prace do wykonania.
- **US-02** Jako uzytkownik chce przenosic zadania miedzy kolumnami, aby odwzorowac postep.
- **US-03** Jako uzytkownik chce wyswietlac tablice, aby zobaczyc stan projektu.
- **US-04** Jako uzytkownik chce usuwac zadania, aby porzadkowac tablice.
- **US-05** Jako uzytkownik chce nadawac priorytety zadaniom (High/Medium/Low). *(feature in progress)*
- **US-06** Jako uzytkownik chce filtrowac widok po priorytecie. *(planned)*

## Definition of Done

- Kod przechodzi lintera i testy jednostkowe
- Dodano test pokrywajacy nowa funkcjonalnosc
- Zaktualizowano README / BACKLOG
- Code review zakonczone approve
- Merge przez PR do `develop`
